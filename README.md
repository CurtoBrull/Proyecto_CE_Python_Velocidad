# Proyecto_CE_Python_Velocidad

Proyecto Curso Especialización Python - Radar Velocidad

## 🚀 Inicio Rápido

### Opción 1: Script Automático (Recomendado)

```bash
# Desarrollo (con auto-reload)
iniciar.bat

# Producción (máximo rendimiento)
iniciar_optimizado.bat
```

### Opción 2: Instalación Manual

```powershell
# Crear entorno virtual
python -m venv .venv

# Activar entorno virtual (Windows PowerShell)
.\.venv\Scripts\Activate.ps1

# Actualizar pip
python -m pip install --upgrade pip

# Instalar dependencias (incluye optimizaciones)
python -m pip install -r requirements.txt
```

## ⚡ Optimización y Rendimiento

Este proyecto incluye optimizaciones que mejoran el rendimiento en **85-96%**:

- **httptools**: Parser HTTP optimizado en C
- **Pool de conexiones** SQLite configurado
- **Modo WAL** para base de datos
- **Índices** en queries frecuentes
- **Session HTTP reutilizable** en Django
